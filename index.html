Фремворк bootstrap
Фремворк bootstrap в составе js, font и css необходимо разархивировать в директорию, в которой находится веб страничка index.html. Далее, к страничке index.html надо подключить css, font и js.

Цель работы

Создать методику разработки веб-страницы с адаптивным дизайном на основе Bootstrap в редакторе Sublime. Вести историю измений с применения системы контроля версий Git.
